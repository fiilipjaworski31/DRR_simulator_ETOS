Na Windows:
1. Skompilować używając MinGW G++:
```
g++ -o simulator.exe *.cpp
```
2. Uruchomić:
```
simulator.exe
```

Na Linux:
1. Skompilować używając G++:
```
g++ -o simulator *.cpp
```
2. Uruchomić:
```
./simulator
```

W obu przypadkach pliki źródłowe (.cpp i .h) muszą znajdować się w tym samym katalogu co komenda kompilacji.
